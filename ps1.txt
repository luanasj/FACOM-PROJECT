Para criar um script PowerShell (.ps1) que execute um script Python e um script Node.js, cada um em um terminal separado, você pode usar o seguinte código:

powershell
# Executa o script Python em um novo terminal
Start-Process powershell -ArgumentList "python C:\caminho\para\seu\script_python.py"

# Executa o script Node.js em um novo terminal
Start-Process powershell -ArgumentList "node C:\caminho\para\seu\script_node.js"
Aqui está o que cada linha faz:

Start-Process powershell -ArgumentList "python C:\caminho\para\seu\script_python.py": Esta linha abre um novo terminal PowerShell e executa o script Python especificado.

Start-Process powershell -ArgumentList "node C:\caminho\para\seu\script_node.js": Esta linha abre um novo terminal PowerShell e executa o script Node.jsespecificado.

Certifique-se de substituir C:\caminho\para\seu\script_python.py e C:\caminho\para\seu\script_node.js pelos caminhos reais dos seus scripts Python e Node.js.

Para executar o script PowerShell, siga estes passos:

Salve o código acima em um arquivo com a extensão .ps1, por exemplo, executar_scripts.ps1.

Abra o PowerShell.

Navegue até o diretório onde o script .ps1 está salvo:

powershell
cd C:\caminho\para\seu\script
Execute o script:

powershell
.\executar_scripts.ps1
Isso abrirá dois novos terminais, um para o script Python e outro para o script Node.js.Se precisar de mais ajuda, estou aqui!
